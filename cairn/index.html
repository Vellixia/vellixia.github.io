Cairn

The open-source context & reliability layer for AI agents — memory, lean context, guardrails, and multi-device sync in one self-hosted Rust binary.

Rust MCP server Local-first Self-hosted Agent memory ★ 0 View on GitHub →

🧠 Agent memory

Persistent, queryable memory for AI agents — across sessions, devices, and tools.

📦 Lean context

Smart retrieval keeps only what matters in the LLM context window. Lower tokens, sharper answers.

🛡️ Guardrails

Built-in reliability layer — schema validation, output checking, and policy enforcement.

🔁 Multi-device sync

Pick up where you left off — sync state across your laptop, server, and dev environments.

🔌 MCP server

Speaks the Model Context Protocol. Plugs into Claude Code, Cursor, and any MCP-compatible client.

⚙️ One Rust binary

Single self-hosted binary. No Docker, no service mesh — just download, run, configure.

Get started

Try Cairn

Cairn is open-source under Apache-2.0. Run the binary, point your MCP client at it, and start giving your agents memory.
