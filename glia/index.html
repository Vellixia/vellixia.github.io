Glia

AI agent skill synthesizer — local-first, air-gappable, graph+vector+causal retrieval over a community catalog.

Rust AI agents Local-first Self-hosted Air-gapped ★ 0 View on GitHub →

🔌 Local-first CLI + Hub

Rust CLI plus a self-hosted Hub. No cloud required — your data stays on your machine.

🧠 Graph + vector + causal retrieval

Hybrid retrieval combining vector similarity, graph traversal, and causal chains — not just one or the other.

🔒 Air-gappable

Designed to run with zero internet. Embeddings, retrieval, and skill composition all work offline.

🧩 Community skill catalog

Discover, install, and compose agent skills from a shared catalog — or run your own private one.

🗄️ SurrealDB + OpenBao

SurrealDB for the unified graph/vector store, OpenBao for secrets — both self-hostable.

🤖 candle embeddings

Local embeddings via Hugging Face candle — no API keys, no rate limits, no data leaving the host.

Get started

Try Glia

Glia is open-source under Apache-2.0. Install the CLI, point it at your Hub, and start composing agent skills.
